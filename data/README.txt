Explanation of data files:

- dat:      main data frame from the experiment, containing the first fixation during which the sound is played and
            the first saccade immediately afterwards. Contains also various parameters and infor regarding the trial, including 
            time stamps, etc.
- fix:      a (big!) data frame containing all fixations in the experiment (+ various details about them)
- quest:    question accuracy data            
- FD:       local, word-level fixation measures for the target word (e.g., FFD, SFD, GD, etc.)
